\# Image Denoising Autoencoder



This project implements an image denoising system using a UNet-based autoencoder trained on the SIDD dataset.



\## Tech Stack

\- PyTorch (Model Training)

\- Flask (Backend API)

\- Flutter (Frontend Application)



\## Features

\- Real-world image denoising

\- Optional sharpening toggle

\- Save denoised output

\- GPU acceleration support
